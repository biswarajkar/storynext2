Python version used: Python 3.6.3
Training Size: 12.5k Positive and 12.5k Negative Reviews (IMDB1)
Test Size: 151 Hand tagged Texts (76 Positive + 75 Negative)

Please run the file "run_nb.py" to classify using Naive Bayes classification and the results (Precision, Recall and F1) will be  saved to the file “results.out”.
Intermediate files are saved in the “obj” folder as pickle files.
